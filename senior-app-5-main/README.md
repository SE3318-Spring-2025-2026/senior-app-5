## Başlangıç

### Gereksinimler

- Node.js (v18+)
- npm veya yarn

### Kurulum

```bash
# Backend kurulumu
cd backend
npm install

# Frontend kurulumu
cd frontend
npm install
```

### Geliştirme

```bash
# Backend'i başlat
cd backend
npm run dev

# Frontend'i başlat
cd frontend
npm run dev
```
